AdaSoft Deploy Tool

คู่มือเริ่มต้นใช้งานอย่างรวดเร็ว

เชื่อมต่อ Git ไม่ได้?

ถ้าคุณเอา Deploy Tool ไปใช้แล้วเชื่อมต่อ Git ไม่ได้ ไม่ต้องกังวล! เรามีวิธีแก้ไข 3 วิธีให้เลือก

เลือกวิธีแก้ปัญหา (1 วิธีใดก็ได้)

1 ใช้ Auto Fix Script แนะนำ

ให้โปรแกรมตรวจสอบและแก้ไขปัญหาให้อัตโนมัติ (ง่ายที่สุด)

  1. คลิกขวาที่ไฟล์ auto-fix-git.bat
  2. เลือก "Run as administrator"
  3. ทำตามคำแนะนำบนหน้าจอ
Script จะตรวจสอบ Git, PATH, Apache และแก้ไขให้อัตโนมัติ

2 ทำตาม Checklist ทีละขั้น ละเอียดที่สุด

คู่มือแบบ step-by-step ที่อธิบายละเอียดทุกขั้นตอน

เปิด Setup Guide
เหมาะสำหรับคนที่ต้องการเข้าใจทุกขั้นตอน

3 ตรวจสอบระบบด้วย System Check อัตโนมัติ

ให้ระบบตรวจสอบและบอกว่าต้องแก้อะไรบ้าง

เปิด System Check
ตรวจสอบ Git, Config, Path และอื่นๆ อัตโนมัติ

Checklist ก่อนใช้งาน

ติดตั้ง Git แล้ว
Git อยู่ใน PATH
Restart Apache แล้ว
Clone โปรเจ็คแล้ว
แก้ไข config.json แล้ว
ใช้ forward slash (/)
สร้างโฟลเดอร์ Deploy แล้ว
System Check ผ่านหมด

พร้อมใช้งานแล้ว?

เมื่อแก้ปัญหาเสร็จแล้ว ก็เริ่มใช้งาน Deploy Tool ได้เลย!

ยังแก้ไม่ได้?

ติดต่อทีม Development พร้อมส่งข้อมูลเหล่านี้:

  • ผลลัพธ์จาก Command Prompt: git --version และ where git
  • Screenshot จาก System Check
  • ไฟล์ config.json
  • Error message จาก Browser Console (F12)
